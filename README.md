# GameProject-Trashbuster

itch.io page: https://kaitojo.itch.io/trashbuster

How to run the game:
Go to the itch.io page and press "Run the Game".

Controls:
A - Move left
D - Move right
Spacebar - Jump
1, 2, 3, 4 - Choose hotbar
Right mouse button - Vacuum
Left mouse button - Shoot trash
Middle mouse button/Scroll wheel (while aiming down) - Air boost
